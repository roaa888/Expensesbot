1

Create a Google Cloud Project

You need a free Google Cloud project to enable the Sheets API and create credentials.

💡
Google Cloud is free for this use case The Google Sheets API has a generous free tier (300 requests/minute). You will not be charged for normal bot usage.

Instructions

Go to console.cloud.google.com and sign in with your Google account
Click the project dropdown at the top → click "New Project"
Name it finance-bot → click "Create"
Wait ~10 seconds → make sure finance-bot is selected in the top dropdown
🖥️
You should see the GCP Console dashboard with "finance-bot" in the top bar
2

Enable Google Sheets & Drive APIs

You need to enable both APIs for the bot to read and write your spreadsheet.

Enable Google Sheets API

In the left menu → APIs & Services → Library
Search for Google Sheets API
Click on it → click the blue "Enable" button

Enable Google Drive API

Go back to Library
Search for Google Drive API
Click on it → click "Enable"
✅
After enabling both, you'll see them listed under APIs & Services → Enabled APIs
3

Create a Service Account

A service account is like a "robot user" that lets the bot access your Google Sheet programmatically.

Instructions

Go to APIs & Services → Credentials
Click + Create Credentials → choose "Service Account"
Fill in the form:
• Service account name: finance-bot-service
• Service account ID: auto-filled (leave as is)
• Description: Finance bot sheets access
Click "Create and Continue"
On "Grant access" and "Grant users access" steps → click "Continue" and "Done" (skip both)
📋
Save the service account email! After creating, you'll see an email like:
finance-bot-service@finance-bot-XXXXX.iam.gserviceaccount.com
You'll need it in Step 6 to share your sheet.
This is saved only in your browser — used in Step 6
4

Download the JSON Credentials File

This file contains the private key that lets the bot authenticate as your service account.

Instructions

In Credentials, find your new service account in the list → click on it
Go to the "Keys" tab (at the top)
Click "Add Key" → "Create new key"
Select "JSON" → click "Create"
A .json file downloads automatically
Rename it to: google_credentials.json
Move it to your project folder: finance_bot/google_credentials.json
⚠️
Keep this file secret! Never commit google_credentials.json to Git or share it publicly. Add it to your .gitignore file.
# The downloaded file looks like this:
{
"type": "service_account",
"project_id": "finance-bot-XXXXX",

"private_key": "-----BEGIN RSA PRIVATE KEY-----\n...",
"client_email": "finance-bot-service@...", ← SAVE THIS

}
5

Create Your Google Sheet

This is where all your expenses will be logged and the monthly summary will appear.

Instructions

Go to sheets.google.com
Click "Blank spreadsheet" (the big + button)
Name it: Finance Bot — Expenses (click "Untitled spreadsheet" at top)
Copy the Sheet ID from the URL bar:
The URL looks like:
https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms/edit
The highlighted part between /d/ and /edit is your Sheet ID
Should be ~44 characters long

6

Share the Sheet with Your Service Account

The bot needs Editor access to write rows. You grant this by sharing like you'd share with a person.

Instructions

In your Google Sheet, click the "Share" button (top-right, green)
In the "Add people and groups" box, paste your service account email:
finance-bot-service@finance-bot-XXXXX.iam.gserviceaccount.com
Change the permission from "Viewer" to "Editor"
Uncheck "Notify people" (service accounts can't receive emails)
Click "Share"
✅
The service account email should now appear in the "People with access" list with Editor role.
7

Update config.py

Paste your Google Sheet ID into the config file — that's the only thing you need to change.

Open finance_bot/config.py and update this line:

# ── Google Sheets ────────────────────────────────────
GOOGLE_CREDENTIALS_FILE = "google_credentials.json" # ← file is already in the right place
GOOGLE_SHEET_ID = "YOUR_GOOGLE_SHEET_ID_HERE" # ← PASTE YOUR ID HERE
📁
File locations check:
finance_bot/
├── google_credentials.json ✅ downloaded in Step 4
├── config.py ← edit GOOGLE_SHEET_ID now
└── main.py
